Welcome to Virtual Piano!

The ultimate easy-to-use online piano app that everyone can play.

What is virtual piano?
Virtual piano keyboard is perfect when there isn't a real piano keyboard at home or when your piano isn't next to your computer. This simulates a real piano keyboard layout with almost two octaves of 24 keys.

How to use the Virtual Piano keyboard?
Use your computer mouse or keyboard to play the virtual piano keyboard. You can view the corresponding computer keyboard letters by activating the "Keyboard Mapping" feature.

What is Keyboard Mapping?
It is a realistic mapping that emulates a real piano keyboard layout. Mark the checkbox to enable. Please note that we are using an English QWERTY computer keyboard as a reference.
